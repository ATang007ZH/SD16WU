# SD16WU大立热成像模块固件交流

SD16后缀分类：
WU是UVC版；G是串口版


15pin接口定义：
1，2是5V，3，4是地，5是D+，6是D-，11脚swdio，12脚swclk

买个支持SWD的Jlink-OB烧写器就可以玩了。推荐买那个可刷jlink-ob，stlink，daplink 3种固件的仿真下载器。
